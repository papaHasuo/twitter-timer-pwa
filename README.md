# 使いすぎタイマー - モバイル版 (PWA)

スマートフォンでのウェブサイト閲覧時間を管理するProgressive Web App (PWA)です。

## 機能

### 🚀 主要機能
- **タイマー機能**: 設定した時間でサイト使用を制限
- **キャラクター警告**: 時間が近づくとかわいいキャラクターが警告
- **サイト監視**: Twitter、Instagram、TikTokなど指定サイトを監視
- **強制休憩**: 時間切れ後は15分間の強制休憩モード
- **通知機能**: プッシュ通知でお知らせ

### 📱 モバイル最適化
- **PWA対応**: スマホにアプリとしてインストール可能
- **オフライン動作**: ネット接続がなくても基本機能が動作
- **レスポンシブデザイン**: あらゆる画面サイズに対応
- **タッチ操作**: スマホでの操作に最適化

### ⚙️ カスタマイズ可能
- **制限時間設定**: 1分〜8時間まで自由に設定
- **監視サイト**: 監視対象サイトを自由に追加・削除
- **キャラクターメッセージ**: 警告メッセージをカスタマイズ
- **通知設定**: プッシュ通知のON/OFF

### 📊 統計機能
- **使用時間追跡**: 日々の使用時間を記録
- **達成率表示**: 目標達成度を可視化
- **連続達成記録**: 連続で目標達成した日数を表示

## インストール方法

### 1. PWAとしてインストール
1. Chrome, Safari, Edgeなどのブラウザで `index.html` を開く
2. ブラウザのメニューから「ホーム画面に追加」を選択
3. アプリ名を確認して「追加」をタップ
4. ホーム画面にアプリアイコンが追加されます

### 2. ローカルサーバーで起動
```bash
# Python 3の場合
python -m http.server 8000

# Node.jsの場合
npx serve .

# その後 http://localhost:8000 にアクセス
```

## 使用方法

### 基本的な使い方
1. **タイマー設定**: 制限時間を分単位で設定
2. **監視サイト追加**: 制限したいサイトのドメインを追加
3. **開始**: タイマーを開始して使用時間を監視
4. **警告対応**: 時間が近づいたらキャラクターの指示に従う

### 設定画面
- **⏰ タイマー**: タイマーの開始・停止・リセット
- **⚙️ 設定**: 制限時間、監視サイト、メッセージの設定
- **📊 統計**: 使用時間や達成率の確認

## 技術仕様

### PWA要件
- **Service Worker**: オフライン機能とキャッシュ管理
- **Web App Manifest**: アプリとしての振る舞いを定義
- **Responsive Design**: モバイルファーストなデザイン

### 対応ブラウザ
- Chrome 67+
- Firefox 62+
- Safari 11.1+
- Edge 79+

### ストレージ
- **localStorage**: 設定とデータの永続化
- **Cache API**: アプリリソースのキャッシュ

## 制限事項

### スマホブラウザでの制約
- **完全なサイトブロック**: ブラウザAPIの制限により、他のタブを完全にブロックはできません
- **バックグラウンド監視**: アプリがバックグラウンドにある際の監視に制限があります
- **クロスドメイン制限**: セキュリティ上の理由で他サイトへの完全な干渉はできません

### 推奨使用方法
1. **セルフコントロール**: 基本的にユーザーの自制心に依存します
2. **通知活用**: プッシュ通知を有効にして意識的な制御を促します
3. **統計活用**: 使用時間の可視化で行動変容を促します

## カスタマイズ

### メッセージのカスタマイズ
`app.js` の `defaultSettings.messages` 配列を編集することで、デフォルトメッセージを変更できます。

### デザインのカスタマイズ
`styles.css` を編集することで、アプリの外観をカスタマイズできます。

### 機能の拡張
`app.js` に新しいクラスメソッドを追加することで、機能を拡張できます。
